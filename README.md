# Favone's Birthday Website - HTML Version

This is the converted normal web version of the Expo React Native app.

## Files

- `index.html` = page structure
- `style.css` = design, fonts, colors, layout, animations
- `script.js` = click actions, gallery creation, flip card, screen switching

## How to run

Open `index.html` in your browser.

For best testing:
1. Open Chrome
2. Right click → Inspect
3. Click mobile icon
4. Select iPhone / Pixel size

## Important

This keeps the same general style:
- priyanka is demo name you can change with your one 
- dark wine background
- Playfair Display, Outfit, Caveat fonts
- golden/pink colors
- polaroid gallery
- flip memory cards
- floating particles
- mobile-first layout
